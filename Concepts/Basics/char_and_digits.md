# char 산술과 자리수 추출

## 핵심 개념

### 1. char와 ASCII

C++에서 `char`는 내부적으로 **정수(숫자)**로 저장된다.
`'a'`는 ASCII 값 **97**, `'b'`는 98, ..., `'z'`는 122.

```cpp
char c = 'a' + 2;   // 97 + 2 = 99 = 'c'
cout << c;          // 출력: c
```

숫자(0~9) → 알파벳(a~j) 변환:

```cpp
int digit = 3;
char ch = 'a' + digit;   // 'a'(97) + 3 = 'd'(100)
cout << ch;              // 출력: d
```

| digit | 'a' + digit | 결과 |
|-------|-------------|------|
| 0     | 97          | 'a'  |
| 1     | 98          | 'b'  |
| 2     | 99          | 'c'  |
| 3     | 100         | 'd'  |
| ...   | ...         | ...  |
| 9     | 106         | 'j'  |

string에 char를 추가할 때는 `(char)` 캐스팅이 필요할 수 있다:

```cpp
string s = "";
s += (char)('a' + 3);   // s = "d"
```

### 2. 각 자리 숫자 추출

`%`와 `/`를 조합해서 정수의 각 자리를 꺼낼 수 있다.

| 연산      | 의미           | 예 (n=23) |
|-----------|----------------|-----------|
| `n % 10`  | 1의 자리 숫자  | 3         |
| `n /= 10` | 1의 자리 제거  | 2         |
| `n % 10`  | 다음 1의 자리  | 2         |

```cpp
int n = 23;
while (n > 0)
{
    int digit = n % 10;   // 마지막 자리 꺼내기
    cout << digit << "\n";
    n /= 10;              // 자리 제거
}
// 출력: 3, 2  (뒤에서부터 나온다!)
```

**주의:** `% 10`은 항상 **낮은 자리부터(역순)** 꺼낸다.

---

### 3. 역순 문제 해결법

숫자를 문자열로 변환할 때, `% 10`이 뒤에서 꺼내므로 순서를 맞춰야 한다.

**방법 1 — 앞에 붙이기 (prepend)**

```cpp
string answer = "";
int age = 23;
while (age > 0)
{
    answer = (char)('a' + age % 10) + answer;   // 새 문자를 앞에 붙인다
    age /= 10;
}
// answer = "cd"
```

**방법 2 — 뒤에 붙인 다음 reverse**

```cpp
#include <algorithm>

string answer = "";
int age = 23;
while (age > 0)
{
    answer += (char)('a' + age % 10);
    age /= 10;
}
reverse(answer.begin(), answer.end());
// answer = "cd"
```

방법 1이 더 간결하다.

### 4. 자리수별 직접 접근 (비교용)

```cpp
// 자리수별로 if를 쓰는 방식 — 동작하지만 확장성이 없다
if (age / 100 != 0)
    answer += (char)('a' + age / 100);
// ... 반복
```

while 루프는 자리수와 관계없이 동작하므로 훨씬 범용적이다.
